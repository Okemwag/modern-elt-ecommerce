# Dashboard

Build the dashboard from gold-layer Databricks models.

Recommended first dashboard pages:

- Executive overview from `mart_monthly_revenue`
- Product performance from `mart_top_products` and `mart_category_performance`
- Customer insights from `mart_repeat_customers` and `dim_customers`
- Regional performance from `fct_orders` and `dim_regions`

Recommended KPIs:

- Total revenue
- Total orders
- Unique customers
- Average order value
- Repeat customer rate
- Payment success rate
- Cancellation rate
- Return rate

Add dashboard files or exported screenshots here after building them in Power BI, Tableau, or Metabase.
