# Dashboard Mockups

Use this as the build brief for Power BI, Tableau, or Metabase.

## Page 1: Executive Overview

Filters:

- Date range
- Region
- Category

Cards:

- Total revenue
- Total orders
- Total customers
- Average order value
- Payment success rate

Charts:

- Monthly revenue trend
- Orders by status
- Revenue by region

## Page 2: Product Performance

Filters:

- Date range
- Category
- Brand

Charts:

- Top products by revenue
- Top products by units sold
- Category revenue
- Product margin proxy using `list_price - unit_cost`

## Page 3: Customer Insights

Filters:

- Date range
- Country
- Repeat customer flag

Charts:

- Top customers by lifetime value
- Repeat vs one-time customers
- Customer count by country
- Last purchase recency bands

## Page 4: Regional Performance

Filters:

- Date range
- Region
- Country

Charts:

- Revenue by region
- Order volume by region
- Cancellation rate by region
- Return rate by region
